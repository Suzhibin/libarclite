Xcode 固化私有组件报错：clang: error: SDK does not contain 'libarclite' at the path xxx (可归类为：缺失 libarclite_iphoneos.a)

使用访达，文件前往：

/Applications/Xcode.app/Contents/Developer/Toolchains/XcodeDefault.xctoolchain/usr/lib/

拷贝整个arc 文件夹到自己的对应位置
